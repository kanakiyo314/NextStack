# NextStack
**Spigot Plugin**

## 機能
- ブロック設置中にスタックがなくなったとき、インベントリから新たに同じスタックをアイテムスロットに設定
- ユーザ毎にプラグインのオン・オフ切り替え

## Commands
- `nextstack < on | off >` プラグインのオン・オフ切り替え

## Permissions
- `ns` - プラグインの有効化・無効化

## Installation
`NextStack-*.jar`ファイルをプラグインディレクトリにコピー
